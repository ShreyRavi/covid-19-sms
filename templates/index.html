COVID-19 SMS Update Service

Welcome to Shrey Ravi's COVID-19 SMS Update Service's homepage, powered by Twilio. To use, please text +1(231) 774-2545 with a zipcode, a City, State Code (such as Chicago, IL), or a state (such as IL or Ohio) to get the latest COVID-19 updates for that area. You can call +1(231) 774-2545 to get these instructions read out loud.
